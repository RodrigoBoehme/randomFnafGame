## Controls
On version 1.0 the game runs on clicking oneof the buttons

Version 2.0 runs similar to the original fnaf 1 game with one or 2 twists, you can click the buttons with the mouse or use the shortcuts as it follows:
Q -> Starts the game
D -> Cams (Check Animatronics current room)
A -> Close and open door (Closed door is visible from the button turning green)
S -> Changes the fuse in case the door breaks
